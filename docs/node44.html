CoordinateTransformation Class

The CoordinateTransformation class handles transformations between custom coordinate systems and latitude-longitude-based systems. Attributes include:

translation: tuple
Translation vector as .
rotation: list or tuple
Rotation matrix or Euler angles.
epsg_code: int
EPSG code for the target coordinate system.
scaling: float or tuple
Optional scaling factors.
reference_elevation: float
Reference elevation for depth conversions.
